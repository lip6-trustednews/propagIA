# Repository for the *propagIA* dataset

The PropagIA dataset is a collection of articles in french, gathered over the year 2024 from 84 distinct sources. The sources were flagged by Viginum as future propaganda generated by genAIs.

## Details

This regroups 2646 articles from 84 sources. The collection window is between dec. 1st 2024 and dec. 1st 2025, with the earlist collected article dated from 2024-12-08 and the latest from 2025-12-01. 

## Structure

Our dataset is provided as a single JSON file. 
Each row represents one article and includes the following fields:

| Field     | Description                         |
| --------- | ----------------------------------- |
| `source`  | Name of the publishing website      |
| `url`     | Direct link to the original article |
| `date`    | Publication date                    |
| `title`   | Article title                       |
| `content` | Article text                        |
